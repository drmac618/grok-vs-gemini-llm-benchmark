# Dataset Dictionary

## benchmark_dataset.csv / benchmark_dataset.json

- **PromptID**: Unique identifier for each benchmark item
- **Category**: Broad content category
- **TaskType**: Type of task being evaluated
- **Difficulty**: Estimated task difficulty
- **SkillTested**: Primary skill the prompt is intended to measure
- **GroundTruthAvailable**: Whether a reference answer is objectively available
- **Prompt**: Benchmark prompt text
- **ExpectedAnswer**: Reference answer used as the benchmark target
